## Описание программы 
*GUI-приложение «GitHub User Finder» с использованием GitHub API, сохранением избранных пользователей и Git.* --- 
## Требования для запуска 
- Python 3.8+
- Tkinter (обычно входит в стандартную поставку Python)
 --- 
## Как запустить 
python main.py 
##Как пользоваться 
- Приложение использует GitHub Search API: GET https://api.github.com/search/users?q={query}
- При необходимости можно добавить Personal Access Token: установите переменную окружения GITHUB_TOKEN и измените запросы, добавив заголовок Authorization: token <GITHUB_TOKEN>.
##Примеры использования:
- Поиск «torvalds» — покажет аккаунт Linus Torvalds.
- Добавление пользователя в избранное: выбрать результат → нажать «Добавить в избранное».
- Показать избранное: кнопка «Показать избранное» (favorites.json хранит список).

##Проверки:
Примеры использования / тесты:
- Попробуйте искать пустую строку — получите предупреждение.
- Найдите пользователя, добавьте в избранное, закройте программу, откройте снова — избранный должен остаться (если favorites.json не в .gitignore).
- Попробуйте удалить пользователя, который не в избранном — получите сообщение.
